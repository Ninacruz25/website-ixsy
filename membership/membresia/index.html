REQUISITOS

Requisitos para la membresía

1.CONTAR CON NÚMERO DE SISTEMA AUTÓNOMO (ASN)
2.CONTAR CON TRANSPORTE HASTA EL IXP
3.SOLICITAR A APTELEKOM O RH EL SITE MAYA O LA CROSS
4.SEGUIR EL PROCESO PARA CONVERTIRSE EN ASOCIADO

Proceso de membresía

1.

PRESENTAR UNA CARTA DE INTENCIÓN PARA PARTICIPACIÓN DEL CONSEJO DIRECTIVO

2.

ESPERAR LA APROBACIÓN DE LA PROPUESTA DE PARTICIPACIÓN

3.

FIRMAR EL CONVENIO DE ADHESIÓN TRAS LA NOTIFICACIÓN DE APROBACIÓN

4.

ACEPTAR LAS POLÍTICAS DE PEERING Y ROUTE SERVER DE IXSY

5.

ESTABLECER LA CONEXIÓN DE TRANSPORTE A IXSY

6.

CONFIGURAR UNA SESIÓN DE BGP CON LOS ROUTE SERVERS DEL IXP EN UNA VLAN ESPECÍFICA O UNA SESIÓN IP

7.

REGISTRARSE EN PEERINGDB

8.

IMPLEMENTAR FILTROS PARA EVITAR ROUTE LEAKS

9.

REALIZAR EL PAGO MENSUAL POR EL USO DEL PUERTO DE 10GBPS O 100GBPS

10.

PAGAR LA MEMBRESÍA ANUAL

11.

PARTICIPAR EN MANRS BAJO EL PROGRAMA DE OPERADORES DE RED

NUESTROS SERVICIOS

IXP Manager

Es una plataforma de administración para puntos de intercambio de Internet (IXP), que incluye gestión de administración completa, gestión gráfica desde mediciones de tráfico de red hasta control de enrutamiento.

IPv6

Certificación de conectividad IPv6 ha permitido la consolidación del IPv6 2023 que se ha vuelto un componente crecimiento de protocolos mejorados para mitigar riesgos de conectividad.

Route Server

Sirve una alternativa de servicios de una Bird (Berkeley Internet Name Domain) que funciona para cualquier usuario que implemente sus propias medidas de ruta.

Nagios

Nagios es un sistema ampliamente utilizado por la industria y es un sistema de monitoreo de red de código abierto que vigila hosts y servicios, alertando a los usuarios cuando las cosas van mal y de nuevo cuando se resuelven.

RPKI

Permite al uso Routing de recursos numericos de registros Reglas aplicables para verificar que los BGP pueden originar un prefijo en BGP están registrados para ser originados desde ASN.

FORT

Router FORT Validator para la familia de protocolos de enrutamiento BGP (Border Gateway Protocol). Es un validador que se ejecuta junto con los enrutadores BGP para verificar las rutas.

MANRS

Proporcionan información sobre la seguridad del enrutamiento con datos sobre protocolos que es información de los miembros que publican en IRR (IRR implementaran en sus propias redes locales.
